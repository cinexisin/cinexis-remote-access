# Cinexis Remote Access — Home Assistant Add-on

Secure remote access to your Home Assistant via **Cinexis Cloud**.

- ✅ No port forwarding required
- ✅ No dynamic DNS needed
- ✅ Works behind CGNAT
- ✅ Encrypted tunnel (TLS)
- ✅ Your HA URL: `https://{your-id}.ha1.cinexis.cloud`

## Installation

1. Add this repository to Home Assistant:
   **Settings → Add-ons → Add-on Store → ⋮ → Repositories**
   ```
   https://github.com/cinexisin/cinexis-remote-access
   ```
2. Install **Cinexis Remote Access**
3. Start the add-on
4. Check the logs for your node ID and URL
5. Contact Cinexis support to activate your node
